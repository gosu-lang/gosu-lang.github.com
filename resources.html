Gosu Resources

Google Newsgroup

Need help getting your gosu program off the ground? Want to discuss features in the next release? Subscribe to the gosu-lang Google group:

  • groups.google.com/group/gosu-lang

Bugs

You can report bugs on Github:

  • https://github.com/gosu-lang/gosu/issues

Examples

We have a few small example projects available on Github:

  • https://github.com/gosu-lang/gosu-example-dynamictype
  • https://github.com/gosu-lang/gosu-example-servlet

Related Projects

Here are a few Gosu-related projects:

  • Ronin - A Gosu-based web framework
  • Aardvark - A Gosu-based extension of the Apache Ant build tool.
  • Goson - A Gosu library for working with JSON.
