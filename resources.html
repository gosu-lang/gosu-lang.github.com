Gosu Resources

Gosu Support

Need help getting your program off the ground? Want to discuss features in the next release, among both users and the Gosu team? Participate in our discussion forum or report a bug:

Forum: http://groups.google.com/group/gosu-lang

Issue tracking: https://github.com/gosu-lang/gosu/issues

Gosu IntelliJ Plugin Support

We also have set of channels for direct access to our IDE developers:

Forum: http://groups.google.com/group/gosu-idea

Issue tracking: https://github.com/gosu-lang/gosu/issues

Related Projects

Here are a few Gosu-related projects:

  • SparkGS - A Gosu-based web micro framework
  • Goson - A Gosu library for working with JSON in a type-safe way.
  • Ronin - A Gosu-based web framework
  • Aardvark - A Gosu-based extension of the Apache Ant build tool.
