Hey look! It's a pragmatic language for the JVM... A simple JVM language. (Note most of the benefits Gosu provides are now available directly in Java via the Manifold Java compiler plugin.) 


  // Type inference keeps things clean
  var aString = "Hello Gosu!"

  // Lists & Maps literals  for easy use
  var strings = { "red",
                  "green",
                  "blue" }

  // Closures make data structures fun
  var l = strings.where( \ s -> s.length() > 3 ).sort()

  // Full Java Interoperability
  var f = new File("Tmp.txt")

  // With new features that make java fun again
  f.write(l.join(", "))
  print( f.read() )
