Released December 2017, 1.14.7 is the latest stable release. See also past releases: 1.5 and earlier or 1.6+ (Beginning with 1.6 we release to Maven Central)

The Quickstart page will help you get up and running with Gosu in no time.

See the Getting Started documentation for several how-to examples, including Gosu on the command line, IDE (Jetbrains IntelliJ IDEA) and popular build tools (Maven and Gradle).

To use Gosu from the command line, just unzip the archive. Be sure you have at least Java JDK 1.8 installed. A valid JAVA_HOME environment variable to this JDK is required.
