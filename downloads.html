Released September 2015, 1.8 is the latest stable release. See also past releases: 1.5 and earlier or 1.6+ (Beginning with 1.6 we release to Maven Central)

See the Getting Started documentation for several how-to examples, including Gosu on the command line, IDE (Jetbrains IntelliJ IDEA) and popular build tools (Maven and Gradle).

To use Gosu from the command line, just unzip/extract one of the below and add the bin directory to your PATH. Be sure you have at least Java 1.8 installed. A valid JAVA_HOME environment variable to this JRE is recommended.

  • gosu-1.8-full.zip
  • gosu-1.8-full.tar.gz

IntelliJ IDEA 14.1 Plugin

Nobody likes writing code without good editor support. Download the Gosu plugin for IntelliJ IDEA 14.1 for access to all of the great features when writing Gosu, including:

  • Code completion
  • Syntax highlighting and error highlighting
  • Go to Declaration, Class, Symbol
  • Rename and Move refactorings
  • Parameter information
  • Find Usages
  • Code formatter
  • Code inspections and intentions
  • Support for Gosu programs
  • Debug Gosu
  • Gosu scratchpad
  • Incremental Gosu compiler
  • Custom typeloader support

To download the plugin for IntelliJ IDEA 14.1, follow these steps:

  1. Install IntelliJ IDEA 14.1 (either the Ultimate or Community editions) to your machine
  2. With IntelliJ running, open the Settings menu and select Plugins
  3. Click the Browse Repositories... button, and search for Gosu
  4. Right click on the Gosu plugin and select Download and Install
  5. After the plugin has been installed, restart IntelliJ

Alternatively, the plugin may be manually downloaded from the Jetbrains plugin repository.

  1. Install IntelliJ IDEA 14.1 (either the Ultimate or Community editions) to your machine
  2. Download the latest version from Jetbrains
  3. With IntelliJ running, open the Settings menu and select Plugins
  4. Click the Install plugin from disk... button, and select the zip file from step two
  5. After the plugin has been installed, restart IntelliJ
