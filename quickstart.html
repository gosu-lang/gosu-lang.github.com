Gosu Quickstart

The Gosu editor is the easiest way to experiment with Gosu:

  • Download the latest Gosu distribution
  • Set the JAVA_HOMEenvironment variable to JDK 1.8's home, if needed
  • Unzip the distribution zip, go to the bin folder and double click on gosu.cmd (or gosu if you are using Linux/Mac)

You can try the bundled example projects like the Match 3 game. Just select Match3 from the Examples pane and run the game by pressing F5.
